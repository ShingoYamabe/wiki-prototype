# 研究室へようこそ

SCR Lab（システム制御・ロボティクス研究室）へようこそ！このWikiは研究室の知識・活動記録を一元管理するための場所です。

## この Wiki の使い方

| やりたいこと | 方法 |
|---|---|
| ページを読む | 左サイドバーからカテゴリを選択 |
| ページを検索する | ヘッダの検索バーに入力（または `/` キー） |
| ページを作成する | `python tools/new_page.py` を実行 |
| ページを編集する | GitHub のウェブエディタ、またはローカルエディタ |

## まず最初にやること

1. **環境構築** → [環境構築ガイド](./environment-setup) を参照してください。
2. **Slack に参加** → 研究室の Slack ワークスペースに参加してください。
3. **GitHub アカウント** → このリポジトリへのアクセス権をメンター・先輩に依頼してください。

## 研究室のルール

{% callout(type="info", title="Wiki 編集について") %}
このWikiはメンバー全員が編集できます。誤り・情報の陳腐化を見つけたら積極的に更新してください。
Pull Request レビューは任意ですが、大きな変更は PR を推奨します。
{% end %}

{% callout(type="warning", title="秘匿情報の取り扱い") %}
アクセストークン・パスワード・個人情報などをWikiに書き込まないでください。
秘匿情報は GitHub Secrets または研究室の共有パスワードマネージャーで管理してください。
{% end %}
